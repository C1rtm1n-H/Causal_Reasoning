# lec3
### do-算子
- do(T=t)表示通过外部干预，强制将总体中的每一个个体的原因变量T的值都设置为t。
    - T的取值不再受其他变量的影响，在因果图上就是删除所有指向T的箭头。
- 因果效应差（causal effect difference）/平均因果效应（average causal effect）：$$ P(Y|do(T=t)) - P(Y|do(T=t')) $$
- 计算带有do算子的概率分布：删除原始因果图中所有指向T的箭头，得到一个新的因果图G'（操纵因果图），在G'上计算条件概率分布P(Y|T=t)（操纵概率）。
    1. 讲因果图转化为操纵图（do定义）
    2. 用条件全概率公式
    3. 利用操纵图中的d-分离（d-分离意味着条件独立）
    4. $ P(Y=y|do(T=t)) = \sum_{Z} P(Y=y|T=t,Z=z)P(Z=z) $
- 举例：辛普森悖论

### 因果发现
- 主流因果发现方法：

| 方法 | 核心思想 | 代表算法 |
| :--- | :--- | :--- |
| 基于约束 | 条件独立性检验推断因果结构 | PC算法 |
| 基于评分 | 搜索因果图结构使得数据的似然函数最大化 | GES算法 |
| 基于功能关系 | 利用变量之间的函数关系推断因果结构 | LiNGAM算法 |

- 基于约束的两个关键假设：
    1. Markov假设：每个变量只与其直接原因变量相关，与非直接原因变量条件独立。
    2. Faithfulness假设：数据中不存在偶然的条件独立关系

PC算法：

- 步骤：

1. 骨架学习：
    1. 初始化：从一个完全连接的无向图开始。
    2. 对每对变量(X,Y)，从空集开始，逐渐增加条件集的大小，进行条件独立性检验。如果在某个条件集S下，X和Y条件独立，则删除边(X,Y)，并记录S作为分离集。（为什么条件集从小到大？-由易到难。）
2. 边的定向：

        1. 规则1：避免新的V结构：如果存在X->Z-Y且X和Y不相连，则定向为Z->Y。
        2. 规则2：避免环路：如果存在X->Z->Y且X和Y不相连，则定向为X->Y。

- 局限性：
    1. 只能识别到Markov等价类，有些边的方向无法仅从观测数据中确定

    3. 忠实性假设可能不满足

### 因果
- 定义：T导致了Y当且仅当保持其他所有变量保持不变的情况下，改变T发现Y也发生了变化。
- 因果效应：T在改变一个单位的时候，Y的变化量
- 因果效应估计：
    - 干预变量：T=1或T=0
    - 潜在结果变量：Y(T=1)或Y(T=0)
    - 反事实问题：每个个体只能观察到一个潜在结果，另一个潜在结果是反事实的。
- 解决反事实问题的方法：
    1. 理想方案：存在平行世界
    2. 随机对照试验：随机选取一部分个体接受干预，另一部分个体不接受干预，比较两组的平均结果。
        - 随机可以使两个组独立同分布，消除混杂因素的影响。
        - 单次结果不可预知，但平均结果可预知。
        - 随机试验的设计：
            1. 样本选择
            2. 随机化设计：完全随机、分层随机、区组随机
            3. 对照组设计
        - 通过随机分配平衡干扰因素，有效避免选择偏倚。
    3. 随机试验的局限性：
        1. 耗时费钱
        2. 伦理问题
        3. 样本有限
- 如果没有随机对照试验，如何估计因果效应？
    - 对观测数据进行调整：通过统计方法控制混杂因素的影响，如回归分析、倾向得分匹配等。

### 典型因果推断方法
1. 匹配法（平行世界简单版本）
    - 核心思想：对于每个接受干预的个体，找到一个或多个在其他特征上相似但未接受干预的个体进行比较。
    - 方法：通过约束混杂变量X相似或一样（$ Distance(X_i, X_j) < \epsilon $）
        1. 精确匹配：要求所有特征完全匹配，适用于离散特征。
        2. 近似匹配：允许特征有一定差异，适用于连续特征。
            - 当混杂变量是高维时，匹配变得困难
    - 局限性：
        1. 难以找到完全匹配的个体
        2. 可能存在未观测的混杂因素
        3. 匹配质量依赖于选择的特征和距离度量
2. 基于倾向得分的方法
    1. 倾向得分匹配：先估计每个个体接受干预的概率（倾向得分$ e(X) = P(T=1|X) $），然后根据倾向得分进行匹配。
        - 倾向得分足以总结混杂变量信息，保证因果推断的无偏性
        - 倾向得分无法被观测到，需要评估
    2. 倾向得分逆加权法：
        - 核心思想：通过给每个个体赋予一个权重来调整样本，使得接受干预和未接受干预的个体在混杂变量上达到平衡。
        - 权重计算：$ w_i = \frac{T_i}{\hat{e}(X_i)} + \frac{1-T_i}{1-\hat{e}(X_i)} $
        - 局限性：
            1. 倾向得分模型的正确性
            2. 极端倾向得分导致权重过大，增加估计的方差
    3. 双稳健算法：
        - 结果变量回归：先建立一个模型预测结果变量Y，得到预测值$\hat{Y}$。
        - 倾向得分逆加权：计算倾向得分并进行逆加权。
        - 结合两者：通过结合结果变量回归和倾向得分逆加权的方法来估计因果效应，即使其中一个模型不正确，仍然可以得到一致的估计。

4. 混杂变量直接平衡法：
    - 核心思想：
    - 动机：变量的分布由变量的各阶矩唯一决定
    - 方法：学习样本使得混杂变量的矩在干预组和对照组一致，从而达到直接平衡混杂变量的目的。
    - 基于矩能唯一确定变量分布，可以直接学习样本权重来实现混杂变量，而无需任何模型假设。

    1. 熵平衡法
    2. 近似残差平衡法
    3. 混杂变量区分平衡法

### 复杂数据偏差
- 工具变量（Instrumental Variable, IV）方法：
- 核心思想：利用一个与干预变量相关但不直接影响结果变量的变量作为工具变量，通过工具变量的变化来估计干预变量对结果变量的因果效应。
- 条件：
    1. 相关性：工具变量Z与干预变量T相关，即$ Cov(Z,T) \neq 0 $
    2. 排除限制：工具变量Z不直接影响结果变量Y，即$ Cov(Z,Y|T) = 0 $
    3. 独立性：工具变量Z与结果变量Y之间没有未观测的混杂因素，即$ Cov(Z,U) = 0 $，其中U是未观测的混杂因素。
